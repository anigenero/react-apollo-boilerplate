# react-apollo-boilerplate
A ReactJS web application that has a simple to-do list to demonstrate the interlocking features.

## Installation

```bash
> ./setup.sh
```

## Building

```bash
> npm run generate:locale
> npm run generate:grahql
> npm run build
```

## Running

```bash
> npm start
```
